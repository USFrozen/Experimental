Personal PyGame-CE experiment

## Requirements:
pygame-ce>=2.5.6\
PyTMX>=3.32

*Should* run on all desktop operating systems running Python 3.10 and up\
Has been tested on Linux and Windows

## To install and play:
### 1. Clone the repository
```
git clone https://github.com/USFrozen/Experimental.git
```

### 2. Install
```
cd Experimental
python install.py
```
Note: The installer simply updates pip and then installs Pygame-CE and PyTMX\
If you do not use pip on your system, please manually install using requirements.txt

### 3. Run game
```
python run_game.py
```
or
```
python main.py
```
